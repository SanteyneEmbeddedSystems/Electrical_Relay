# Electrical_Relay

This repository defines a package gathering software elements allowing to manage
electrical relays.

## Content

The Electrical_Relay package gathers :
* Data_Type :
  * E_Relay_Configuration
* Interface :
  * Relay_Actuation_Requests
* Component_Type :
  * Relay_Actuator

sketchbook folder_.
